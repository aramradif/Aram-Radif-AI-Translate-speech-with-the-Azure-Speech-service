# Azure Speech Translator

This project demonstrates **speech-to-text translation** and **speech synthesis** using **Azure AI Speech Services** in Python.

## Features
- Translate English speech to French, Spanish, or Hindi
- Output translation as text
- Synthesize translated speech to a `.wav` audio file

## Requirements
- Python 3.8+
- Azure subscription with Speech resource
- `pip install -r requirements.txt`
